<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| screencast frames → localhost dashboard | JPEG frames of the target's authenticated pages are pushed to the loopback dashboard; they must stay on 127.0.0.1 and never leave the machine. |
| dashboard ↔ playwright/outbound | The dashboard must remain a pure inbound server — no Playwright, no outbound client — so it can never be repurposed to exfiltrate frames. |
| model reasoning → rendered HTML | The model's verbatim reasoning is rendered; it must be injected as text (textContent), never HTML, to stay XSS-safe while remaining verbatim. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-15 | Information Disclosure | screencast frames of authenticated pages leaking off-box | mitigate | The dashboard binds 127.0.0.1 only (existing T-03-09); frames ride the loopback SSE channel; the dashboard imports no outbound client and no playwright (new no-network assertion). |
| T-05-16 | Tampering / Elevation | the dashboard module repurposed to pull frames or call out via Playwright | mitigate | GATE-03 dashboard tier keeps http.request/http.get/fetch/https forbidden here; the new assertion forbids `from 'playwright'` under src/dashboard/. The CDP wiring lives in src/agent/screencast.ts. |
| T-05-17 | Tampering | model reasoning containing markup breaking/injecting the dashboard DOM | mitigate | Reasoning is rendered via textContent (verbatim + XSS-safe), never innerHTML; asserted by source inspection. |
| T-05-18 | Denial of Service | unthrottled screencast frames flooding the SSE channel | mitigate | everyNthFrame throttles to ~2fps and each frame is acked; only the latest frame is cached; a frame-handler throw is swallowed and never halts the run. |

No package installs; no new outbound surface (frames stay on the existing loopback SSE server; the
only outbound anywhere remains the pinned provider endpoint from 05-01). The no-network guard —
now including the dashboard-no-playwright assertion — stays green.
