<objective>
Stop paying for a vision call on every step. A **pure semantic change detector** compares the
current page against the state at the **last model call**; if nothing meaningful changed (route
template, interactive-element kinds, a modal/dialog, or the form-field set), the loop **skips the
vision model** and takes a deterministic frontier policy step instead. The skipped step is a
first-class, honestly-labelled record: its `source` is `'policy'` and its reasoning is the real
deterministic decision line (never a fabricated model rationale — DASH-06's verbatim rule stays
intact). The number of skipped calls surfaces in the spec coverage block and as a dashboard
counter, so the cost saving is visible.

Output: `src/agent/changeDetect.ts` (pure); change-gating + skip accounting in `src/agent/loop.ts`;
a `source` field on agent-step records + `recordModelCallsSkipped` in the store; the count carried
into the spec coverage block; and a dashboard skip counter. All automated tests run on the scripted
provider.
</objective>
