<objective>
Give the autonomous loop a **hard cost ceiling and polite pacing** so an unattended run against a
real account cannot run away, and surface the loop's **stop reason** that 05-05 found was
silently discarded. The lever is real token usage: `Provider.chat` is extended to return
`{ text, usage }`, the scripted provider reports zeros, the anthropic provider parses usage from
its response as a pure function, `decideWithRetry` sums usage across its (up to two) calls, and a
new `BudgetTracker` + static price table turns that into a `--max-tokens` / `--max-cost` ceiling
that stops the run cleanly with `stopReason 'budget'` — **still generating a partial spec**. A
`Pacer` with an injected clock enforces `--pace-ms` between actions. `STOP_REASONS.BUDGET` is
added, and `ExploreResult.stopReason` is now printed at explore end and written into the spec
coverage block.

This plan deliberately **evolves** the fixed `Provider.chat` return contract (string →
`ChatResult`), so the existing model/decision/loop tests are updated in lockstep — a documented
contract evolution, net test count still grows.

Output: `src/agent/{budget,pace}.ts`; `Provider.chat` returning `{ text, usage }` across
`src/model/`; budget + pacing + budget-stop wired into `src/agent/loop.ts`; the stop-reason
surfacing path through `src/capture/store.ts` (manifest) → `src/spec/generator.ts` (coverage) →
`src/cli/explore.ts` (print); and `--max-tokens` / `--max-cost` / `--pace-ms` on `archeo
explore`. All automated tests run on the scripted provider (no key, no network).
</objective>

# src/agent/budget.ts (COST-01/03)
export interface TokenUsage { inputTokens: number; outputTokens: number }   // re-export shape from model/types via structural match
export interface ModelPrice { inputPer1M: number; outputPer1M: number }     // USD per 1,000,000 tokens
export const PRICE_TABLE: Record<string, ModelPrice> = {
  'claude-haiku-4-5':  { inputPer1M: 1.0,  outputPer1M: 5.0  },
  'claude-sonnet-4-6': { inputPer1M: 3.0,  outputPer1M: 15.0 },
  'claude-opus-4-8':   { inputPer1M: 5.0,  outputPer1M: 25.0 },
}   // editable code constant; anthropic models only for now (scripted 'frontier' is absent → cost disabled)
export function priceForModel(model: string): ModelPrice | undefined   // PRICE_TABLE[model]
export function costOf(usage: TokenUsage, price: ModelPrice): number    // (in/1e6)*inputPer1M + (out/1e6)*outputPer1M
export class BudgetTracker {
  constructor(opts: { maxTokens?: number; maxCost?: number; model?: string })
  add(usage: TokenUsage): void
  get totalTokens(): number     // inputTokens + outputTokens accumulated
  get totalCost(): number       // 0 when model absent from PRICE_TABLE (cost tracking disabled)
  exceeded(): boolean           // (maxTokens !== undefined && totalTokens >= maxTokens) || (maxCost !== undefined && cost>0 && totalCost >= maxCost)
}
// NOTE: exceeded uses >= so --max-tokens 0 forces an immediate budget stop even under the
// zero-usage scripted provider (this is exactly the deterministic offline budget proof 06-06 uses).
